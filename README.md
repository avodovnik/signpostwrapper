# signpostwrapper
Wraps the OS Signpost events in a library that can be bound to from a Xamarin app

# Build instructions
` xcodebuild -project SignPostWrapper.xcodeproj -target SignPostWrapper -sdk macosx10.15 -configuration Release clean build `
